# 用户认证系统说明

## 注册功能

### 有效注册码

以下是系统中预设的有效注册码（不区分大小写）：

- `BUNNY2024`
- `WORDS2024`
- `STUDY2024`
- `LEARN2024`
- `HAPPY2024`

### 注册流程

1. 用户访问注册页面 `/register`
2. 填写手机号（必须是有效的中国大陆手机号）
3. 输入有效的注册码
4. 设置密码（8-20位字母和数字组合）
5. 确认密码
6. 点击"注册"按钮

### 密码要求

- 长度：8-20位
- 必须包含字母和数字
- 示例：`abc12345`、`Test1234`

## 登录功能

### 登录流程

1. 用户访问登录页面 `/login`
2. 输入注册时使用的手机号
3. 输入密码
4. 点击"登录"按钮

### 账号有效期

- 注册成功后，账号默认有效期为1年
- 在Header的用户下拉菜单中可以查看到期日期和剩余天数
- 账号过期后将自动登出

## 功能特性

### 登录前

- 主页显示"立即注册"按钮
- 不显示"在学"栏目
- 单词书卡片显示，但不显示学习进度

### 登录后

- 主页不显示"立即注册"按钮
- 显示"在学"栏目
- 单词书卡片显示学习进度
- Header右上角显示用户头像和信息
- 可以查看账号到期时间和剩余天数
- 可以退出登录

## 数据存储

所有用户数据存储在浏览器的 `localStorage` 中：

- `bunny_users`: 所有注册用户的列表
- `current_user`: 当前登录用户的信息

## 测试账号

您可以使用以下信息进行测试：

1. 使用任意有效注册码（如 `BUNNY2024`）注册新账号
2. 手机号示例：`13800138000`
3. 密码示例：`test1234`

## 修改注册码

如需添加或修改有效注册码，请编辑文件：
`/lib/utils/auth-manager.ts`

在 `VALID_REGISTRATION_CODES` 数组中添加新的注册码。
